1. Introducción

Este documento proporciona instrucciones detalladas para instalar y configurar Easy Wallet, un sistema de gestión financiera personal desarrollado en Java con conexión a MySQL.

3. Instalación Paso a Paso

3.1. Instalación de Dependencias

Instalar Java JDK

  1. Descargar desde Oracle JDK
  2. Configurar variables de entorno: 
    export JAVA_HOME=/usr/lib/jvm/jdk-17
export PATH=$PATH:$JAVA_HOME/bin

Instalar XAMPP (para MySQL)

  1. Descargar desde Apache Friends
  2. Ejecutar instalador y activar MySQL en el panel de control

3.2. Configuración de la Base de Datos

  1. Abrir phpMyAdmin (http://localhost/phpmyadmin)
  2. Crear una nueva base de datos: 
    CREATE DATABASE "bank";
  3. Importar el archivo SQL:
    • Seleccionar la base easywallet
    • Ir a Importar > Subir bank(13).sql

3.3. Configuración del Proyecto en NetBeans

  1. Clonar o descargar el repositorio del proyecto
  2. Abrir NetBeans y seleccionar File > Open Project
  3. Asegurar las dependencias:
    • Click derecho en el proyecto > Properties > Libraries
    • Agregar mysql-connector-java-8.0.xx.jar
Importante: No dejar credenciales en blanco en entornos de producción.

4. Verificación de la Instalación

  1. Ejecutar el proyecto en NetBeans (Run > Build Project)
  2. Verificar logs en consola para errores de conexión
  3. Probar inicio de sesión con credenciales predeterminadas (si aplica)
